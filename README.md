# PushToDocs
## Automated Documentation Pusher
PushToDocs is a cutting-edge tool designed to streamline the process of generating and pushing documentation to various platforms. With its robust features and intuitive interface, PushToDocs aims to simplify the documentation workflow, making it easier for developers to maintain up-to-date and accurate documentation.

## Description
PushToDocs is built to cater to the needs of modern software development teams. It provides a seamless way to generate documentation from code comments, track changes, and push updates to documentation platforms. The recent update includes a fix for the groq model, ensuring improved performance and accuracy.

## Features
* Automated documentation generation from code comments
* Support for multiple documentation platforms
* Real-time tracking of changes and updates
* Customizable templates for documentation
* Integration with popular version control systems

## Installation
To get started with PushToDocs, follow these steps:
1. Clone the repository: `git clone https://github.com/SudeepKagi/PushToDocs.git`
2. Install dependencies: `npm install` or `yarn install`
3. Configure environment variables: `cp .env.example .env` and update the values as needed

## Usage
1. Run the documentation generator: `npm run generate-docs` or `yarn run generate-docs`
2. Push updates to documentation platforms: `npm run push-docs` or `yarn run push-docs`
3. Track changes and updates: `npm run track-changes` or `yarn run track-changes`
